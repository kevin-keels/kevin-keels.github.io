# Shelter Management System

A full-stack web application for managing information in an animal shelter, including dogs, adopters, adoption applications, and administrative decisions.

---

## Live Demo
https://shelter-system.onrender.com  
(If the site initially loads slowly, it is waking from free-tier sleep.)

---

## Features
- User authentication and role-based access
- Add, update, and remove dogs and adopters
- View and process adoption applications
- Approve or reject requests with recorded dates
- Clean, dark-mode inspired UI theme

---

## Technology Used
| Component | Technology |
|----------|------------|
| Backend  | Python, Flask |
| Frontend | HTML, CSS, Jinja2 |
| Database | SQLite |
| Hosting  | Render |
